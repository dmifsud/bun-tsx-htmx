# bun-tsx-htmx

To install dependencies:

```bash
bun install
```

To run server and frontend dev together:

```bash
bun run dev
```

# Production build

```bash
bun run build
```
then to test:
```bash
bun run serve
```

This project was created using `bun init` in bun v1.1.19. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
